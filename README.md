# openrgb
OpenRGB rainbow color profile

Tested on Razer Ornata Chroma

## Installation
- Copy `rainbow.orp` file into your openrgb folder config
(ex: ~/.config/OpenRGB/rainbow.orp)

- Load the profile on the application
